= spk_dashboard

* 

== DESCRIPTION:

At the moment a small library and collection of scripts used to load and manipulate Puppet Dashboard data

== FEATURES/PROBLEMS:

* Puppet dashboard lacks at the moment fully functioning API, parts that do function still have bugs.


This task will install any missing dependencies, run the tests/specs,
and generate the RDoc.
